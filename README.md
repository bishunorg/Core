### About

- `almanac`: 一级汉字表，包含注音，笔画等
- `heteronym`: 多音字表，部分多音字，目前共收录 357 个多音字
- `pinyin.md`: 一级汉字表，共 3,500 字，按照字母 A-Z 排序
- `strokes.md`: 一级汉字表，共 3,500 字，按照笔画排序
